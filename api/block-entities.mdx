This page documents the public API for Vista's block entity classes. Block entities store data and handle the logic for blocks with complex behavior.

## TVBlockEntity

**Package:** `net.mehvahdjukaar.vista.common.tv`

**Extends:** `ItemDisplayTile`

The `TVBlockEntity` class manages the state and behavior of TV blocks, including video playback, cassette storage, and enderman interaction detection.

<ParamField path="world" type="Level">
  The level/world
</ParamField>

  The block position
</ParamField>

### Usage Example

```java
// Get a TV block entity
if (level.getBlockEntity(pos) instanceof TVBlockEntity tv) {
    // Check if a cassette is inserted
    ItemStack cassette = tv.getDisplayedItem();
    if (!cassette.isEmpty()) {
        // Get video information
        IVideoSource video = tv.getVideoSource();
        int playbackTicks = tv.getPlaybackTicks();
        boolean paused = tv.isPaused();
        
        System.out.println("Playing video at " + playbackTicks + " ticks");
        if (paused) {
            System.out.println("Video is paused");
        }
    }
    
    // Get TV screen dimensions
    int connectedWidth = tv.getConnectedCount();
    int pixelWidth = tv.getScreenPixelWidth();
    int pixelHeight = tv.getScreenPixelHeight();
    System.out.println("TV is " + connectedWidth + "x" + connectedWidth + " blocks");
    System.out.println("Screen resolution: " + pixelWidth + "x" + pixelHeight);
    
    // Insert a cassette programmatically
    ItemStack newCassette = new ItemStack(VistaMod.CASSETTE.get());
    if (tv.canPlaceItem(0, newCassette)) {
        tv.setDisplayedItem(newCassette);
        tv.setChanged();
    }
}
```

### Usage Example

```java
// Get a view finder block entity
if (level.getBlockEntity(pos) instanceof ViewFinderBlockEntity viewFinder) {
    // Get camera orientation
    float yaw = viewFinder.getYaw();
    float pitch = viewFinder.getPitch();
    int zoom = viewFinder.getZoomLevel();
    
    System.out.println("Camera facing: yaw=" + yaw + ", pitch=" + pitch);
    System.out.println("Zoom level: " + zoom + "/" + ViewFinderBlockEntity.MAX_ZOOM);
    
    // Adjust camera orientation
    ViewFinderAccess access = viewFinder.selfAccess;
    viewFinder.setYaw(access, 45.0f);  // Face northeast
    viewFinder.setPitch(access, -30.0f);  // Look up 30 degrees
    viewFinder.setZoomLevel(10);  // Set moderate zoom
    viewFinder.setChanged();
    
    // Lock the camera
    viewFinder.setLocked(true);
    
    // Get the broadcast ID for linking with hollow cassettes
    UUID broadcastId = viewFinder.getUUID();
    System.out.println("Broadcast ID: " + broadcastId);
    
    // Calculate FOV for this zoom level
    float fov = viewFinder.getFOV();
    System.out.println("Current FOV: " + fov + " degrees");
    
    // Insert a color filter (stained glass pane)
    ItemStack redPane = new ItemStack(Items.RED_STAINED_GLASS_PANE);
    if (viewFinder.canPlaceItem(0, redPane)) {
        viewFinder.setDisplayedItem(redPane);
        viewFinder.setChanged();
    }
}
```

    // Check if player can edit

        // Set a video URL
        projector.setUrl("https://example.com/broadcast.mp4");
        projector.setChanged();
        
        // Update clients
        level.sendBlockUpdated(

    // Get current URL
    String url = projector.getUrl();
    if (!url.isEmpty()) {
        System.out.println("Broadcasting from: " + url);
    }
}

## Supporting Interfaces

### IBroadcastSource

**Package:** `net.mehvahdjukaar.vista.common.cassette`

Interface for block entities that can broadcast video feeds.

```java
public interface IBroadcastSource {
    UUID getUUID();
    @Nullable IVideoSource getBroadcastVideo();
    
    default void ensureLinked(Level level, IBroadcastLocation location);
    default void removeLink(Level level);
}
```

<Accordion title="Interface Methods">
**`getUUID()`** - Returns the unique identifier for this broadcast source

**`getBroadcastVideo()`** - Returns the video source being broadcast, or null

**`ensureLinked()`** - Registers this broadcast source with the BroadcastManager

**`removeLink()`** - Unregisters this broadcast source
</Accordion>

### ViewFinderAccess
