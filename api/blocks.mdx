This page documents the public API for Vista's block classes. These classes define the behavior and properties of blocks that can be placed in the world.

## TVBlock

**Package:** `net.mehvahdjukaar.vista.common.tv`

**Extends:** `HorizontalDirectionalBlock`

**Implements:** `EntityBlock`, `Equipable`, `IOptionalEntityBlock`

The `TVBlock` class represents a television block that can display video content from cassettes. TVs can connect together to form larger displays.

### Properties

<ParamField path="CODEC" type="MapCodec<TVBlock>">
  Codec for serialization and deserialization of TV blocks
</ParamField>

<ParamField path="POWER_STATE" type="EnumProperty<PowerState>">
  Block state property representing the power state of the TV. Values: `OFF`, `DIRECT`, `INDIRECT`
</ParamField>

<ParamField path="CONNECTION" type="EnumProperty<TVType>">
  Block state property representing how this TV connects to adjacent TVs. Values: `SINGLE`, `CENTER`, `TOP`, `BOTTOM`, `LEFT`, `RIGHT`, `TOP_LEFT`, `TOP_RIGHT`, `BOTTOM_LEFT`, `BOTTOM_RIGHT`
</ParamField>

### Constructor

```java
public TVBlock(Properties properties)
```

Creates a new TV block with the specified properties. The block will emit light level 3 when powered.

### Usage Example

```java
// Get the TV block from the registry
Block tvBlock = VistaMod.TV.get();

// Check if a block state is a TV
if (state.getBlock() instanceof TVBlock tv) {
    // Get the power state
    PowerState power = state.getValue(TVBlock.POWER_STATE);
    boolean isPowered = power.isOn();
    
    // Get the connection type
    TVType connection = state.getValue(TVBlock.CONNECTION);
    
    // Get the facing direction
    Direction facing = state.getValue(TVBlock.FACING);
}

// Create a TV block state
BlockState tvState = tvBlock.defaultBlockState()
    .setValue(TVBlock.FACING, Direction.NORTH)
    .setValue(TVBlock.POWER_STATE, PowerState.OFF)
    .setValue(TVBlock.CONNECTION, TVType.SINGLE);
```

## ViewFinderBlock

**Package:** `net.mehvahdjukaar.vista.common.view_finder`

**Extends:** `DirectionalBlock`

**Implements:** `EntityBlock`, `IRotatable`, `IAnalogRotatable`

The `ViewFinderBlock` class represents a camera block that can be controlled by players to view the world from different angles. It can broadcast a live feed to TVs via hollow cassettes.

### Usage Example

```java
// Get the view finder block
Block viewFinderBlock = VistaMod.VIEWFINDER.get();

// Check if a block is a view finder
if (level.getBlockState(pos).getBlock() instanceof ViewFinderBlock) {
    // Get the block entity
    if (level.getBlockEntity(pos) instanceof ViewFinderBlockEntity viewFinder) {
        // Get camera orientation
        float yaw = viewFinder.getYaw();
        float pitch = viewFinder.getPitch();
        int zoomLevel = viewFinder.getZoomLevel();
        
        // Get the unique ID for linking with cassettes
        UUID viewFinderUUID = viewFinder.getUUID();
    }
}

// Place a view finder facing up
BlockState viewFinderState = viewFinderBlock.defaultBlockState()
    .setValue(ViewFinderBlock.FACING, Direction.UP)
    .setValue(ViewFinderBlock.ROTATE_TILE, Rotation.NONE);
```

## SignalProjectorBlock

**Package:** `net.mehvahdjukaar.vista.common.projector`

**Extends:** `BaseEntityBlock`

The `SignalProjectorBlock` is an advanced block that can broadcast video content from URLs. It requires operator permissions or ComputerCraft to edit.

</ParamField>

</ResponseField>

### Usage Example

```java
// Get the signal projector block
Block projectorBlock = VistaMod.SIGNAL_PROJECTOR.get();

// Check if a block is a signal projector
if (level.getBlockState(pos).getBlock() instanceof SignalProjectorBlock) {
    // Get the block entity
    if (level.getBlockEntity(pos) instanceof SignalProjectorBlockEntity projector) {
        // Get the configured URL
        String url = projector.getUrl();
        
        // Set a new URL (requires operator permissions)
        if (projector.canBeEditedBy(player)) {
            projector.setUrl("https://example.com/video.mp4");
            projector.setChanged();
        }
        
        // Get the broadcast UUID for linking
        UUID broadcastId = projector.getUUID();
    }
}
```

## Enums and Supporting Types

### PowerState

**Package:** `net.mehvahdjukaar.vista.common.tv`

Represents the power state of a TV block.

```java
public enum PowerState implements StringRepresentable {
    OFF,      // TV is not powered
    DIRECT,   // TV is directly powered by redstone
    INDIRECT  // TV is indirectly powered (through connection)
}
```

<Accordion title="PowerState Methods">
```java
public static PowerState direct(boolean powered)
```
Returns `DIRECT` if powered is true, `OFF` otherwise.

```java
public static PowerState indirect(PowerState existing, boolean indirectPower)
```
Returns the appropriate indirect power state based on existing state.

```java
public boolean isOn()
```
Returns `true` if the state is not `OFF`.

```java
public boolean isStrong()
```
Returns `true` if the state is `DIRECT`.
</Accordion>

### TVType

**Package:** `net.mehvahdjukaar.vista.common.tv`

Represents how a TV block connects to adjacent TV blocks.

```java
public enum TVType implements StringRepresentable {
    SINGLE,        // Standalone TV
    CENTER,        // Connected on all sides
    TOP,           // Connected below, left, and right
    BOTTOM,        // Connected above, left, and right
    LEFT,          // Connected above, below, and right
    RIGHT,         // Connected above, below, and left
    TOP_LEFT,      // Connected below and right
    TOP_RIGHT,     // Connected below and left
    BOTTOM_LEFT,   // Connected above and right
    BOTTOM_RIGHT   // Connected above and left
}
```

<Accordion title="TVType Methods">

```java
public boolean isConnected(Direction2D dir)
```
Checks if this TV is connected in the given 2D direction.
</Accordion>
