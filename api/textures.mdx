# Texture Management

Vista's texture management system handles live feeds, cassette animations, and web-based video content.

## LiveFeedTexturesManager

Manages textures for live camera feeds from viewfinders.

**Package**: `net.mehvahdjukaar.vista.client.textures`

### Key Features

- **Feed Registration**: Tracks active feeds by UUID
- **Texture Caching**: Reuses textures for the same feed
- **Update Scheduling**: Adaptive update rate based on screen size and distance
- **Performance Tracking**: Monitors update times and frame rates

### Static Fields

<ParamField path="UPDATE_TIMES" type="Map<UUID, RollingBuffer<Long>>">
  Stores update timestamps for each feed UUID for performance monitoring
</ParamField>

<ParamField path="SCHEDULER" type="Supplier<AdaptiveUpdateScheduler>">
  Adaptive scheduler that adjusts update frequency based on performance
</ParamField>

### Usage Example

```java
UUID feedId = UUID.randomUUID();
LiveFeedTexture texture = LiveFeedTexturesManager.getOrCreate(feedId, level, blockPos);

// Texture will automatically update based on scheduler
// Access performance metrics
RollingBuffer<Long> updateTimes = LiveFeedTexturesManager.UPDATE_TIMES.get(feedId);
double averageMs = LiveFeedTexturesManager.SCHEDULER.get().getAverageUpdateTimeMs();
```

## CassetteTexturesManager

Manages animated textures for cassette tapes.

**Package**: `net.mehvahdjukaar.vista.client.textures`

### Features

- **Animation Loading**: Loads GIF and animated texture data
- **Frame Management**: Handles multi-frame animations
- **Texture Atlas**: Organizes cassette textures efficiently
- **Resource Pack Support**: Reloads textures when resource packs change

### Usage Example

```java
// Get animated texture for a cassette tape
Holder<CassetteTape> tape = // ... cassette tape holder
AnimatedStripTexture texture = CassetteTexturesManager.getTexture(tape);

// Texture automatically animates based on game ticks
int currentFrame = texture.getCurrentFrame(ticks);
```

## WebTexturesManager

Manages textures loaded from web URLs.

**Package**: `net.mehvahdjukaar.vista.client.textures`

### Features

- **URL Loading**: Downloads images from web URLs
- **Caching**: Caches downloaded textures
- **GIF Support**: Handles animated GIF files
- **Error Handling**: Provides fallback textures on failure

<Warning>
  Web texture loading requires appropriate server permissions and may have performance implications. Use with caution.
</Warning>

### Usage Example

```java
String url = "https://example.com/video.gif";
WebTexture texture = WebTexturesManager.getOrLoad(url);

if (texture.isReady()) {
    // Texture has finished loading
    ResourceLocation textureLocation = texture.getLocation();
}
```

## AnimatedStripVertexConsumer

Vertex consumer wrapper that applies frame-based UV coordinates for animated strips.

**Package**: `net.mehvahdjukaar.vista.client.textures`

### Constructor

<ParamField path="delegate" type="VertexConsumer" required>
  Underlying vertex consumer
</ParamField>

<ParamField path="texture" type="AnimatedStripTexture" required>
  Animated strip texture
</ParamField>

<ParamField path="ticks" type="int" required>
  Current game ticks for animation
</ParamField>

### Usage

```java
AnimatedStripTexture animTexture = // ... get animated texture
VertexConsumer baseConsumer = buffer.getBuffer(renderType);
VertexConsumer animated = new AnimatedStripVertexConsumer(
    baseConsumer, 
    animTexture, 
    videoTicks
);

// Use animated consumer for rendering
animated.addVertex(poseStack.last().pose(), x, y, z)
    .setColor(255, 255, 255, 255)
    .setUv(u, v)  // UVs automatically adjusted for current frame
    .setNormal(normal.x, normal.y, normal.z);
```
