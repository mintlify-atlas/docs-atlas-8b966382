The **Signal Projector** is a block entity that fetches and broadcasts video content from external URLs. It acts as a video source for the Vista mod's broadcast system.

## Overview

Signal Projectors allow you to display custom video content on TVs by providing a URL. They integrate with the broadcast system and can be linked to hollow cassettes for remote viewing.

<Warning>
  Signal Projectors require operator permissions or ComputerCraft to configure by default.
</Warning>

You can control the Signal Projector programmatically:

```lua

-- Set the video URL
```

## Security Considerations

<Warning>
  Signal Projectors fetch content from external URLs. Be cautious about:
  
  - **Bandwidth**: Streaming video can consume significant bandwidth
  - **Content**: Ensure URLs point to appropriate content
  - **Permissions**: Restrict who can configure projectors on servers
  
  Consider using the operator-only permission in multiplayer environments.
</Warning>

## Comparison with Other Sources

| Feature | Signal Projector | Viewfinder | Cassette |
|---------|------------------|------------|----------|
| **Content Type** | External URL | Live camera feed | Pre-recorded |
| **Configuration** | URL input | Rotation/zoom | Fixed content |
| **Permissions** | Op/CC required | Anyone | Anyone |
| **Broadcasting** | Yes | Yes | Direct insertion |
| **Use Case** | Custom content | Surveillance | Music/videos |

Signal Projectors are best for custom, external content that can be updated without changing game files.
