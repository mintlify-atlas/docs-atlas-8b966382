Cassette Tapes are pre-recorded video items that can be inserted into [Televisions](/blocks/television) to play various animated videos with accompanying music.

## Overview

Cassettes display custom animated videos when inserted into a powered Television. Each cassette has a unique video, color, and some feature music tracks created by various artists.

<Card title="Obtaining Cassettes" icon="treasure-chest">
  Cassettes can be found as rare treasure loot in dungeon chests, or dropped by Creepers when killed by a Pillager.
</Card>

## Using Cassettes

### Playing on TVs

1. Place a [Television](/blocks/television) and power it with redstone
2. Right-click the TV with a cassette to insert it
3. The video will automatically start playing when powered
4. Sneak + right-click the TV (while holding no cassette) to pause/unpause playback
5. Right-click the TV with an empty hand or another cassette to eject the current cassette

### Redstone Output

Cassettes provide a redstone comparator signal when inserted into a TV. Each cassette has a different signal strength, allowing you to detect which cassette is playing:

## Custom Supporter Cassettes

<Note>
Supporters of the mod can create custom cassettes by crafting a cassette and renaming it on an anvil to specific names.
</Note>

The mod includes special supporter cassettes that can be unlocked by renaming a blank cassette to one of these names:

- **zam** - Blue themed supporter cassette (#494ef1)
- **little_pianist** - Light colored cassette (#fffddf)
- **leftchaotix** - Cyan themed supporter cassette (#2ad6f0)
- **color_test** - Test pattern cassette (#aabbcc)

### How to Create Supporter Cassettes

1. Obtain any regular cassette
2. Place it in an anvil
3. Rename it to one of the supporter names (case-sensitive)
4. Take it out - it will automatically transform into the supporter cassette

<Warning>
The custom name must exactly match the supporter name (all lowercase) for the transformation to work.
</Warning>

## How Cassettes are Obtained

### Treasure Loot

Cassettes have a 3% chance to appear in treasure chests found in:
- Dungeons
- Mineshafts
- Strongholds
- Desert Temples
- Jungle Temples
- End Cities
- Other structure loot chests

### Creeper Drops

When a Creeper is killed by a Pillager's arrow, it has a chance to drop a random cassette tape.

## Tips

<Tip>
Connect multiple TVs together (3 or more) to create a large screen display and earn the **Absolute Cinema** advancement when playing a cassette!
</Tip>

<Tip>
Use the different comparator output values to create redstone circuits that respond to specific cassettes being played.
</Tip>
